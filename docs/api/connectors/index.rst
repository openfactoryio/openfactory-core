OpenFactory Connectors
======================

OpenFactory defines Connectors which which translate between industrial protocols
(e.g., OPC UA, Modbus, MTConnect) and the OpenFactory data model.

All Connectors derive from :class:`openfactory.connectors.base_connector.Connector` class.
